# BackpackPlus-AldisClassesCompat
Adjusts the sizes of the backpacks across all of vanilla/Backpackpack to be more consistently upgraded as you go up in tech tiers, and secondly it's a compatibility patch between Aldi's Classes and BackpackpackPlusFG, making the 'Bigger' versions actually bigger, also adjusts the progression of backpacks to make each upgrade feel better, imo.
